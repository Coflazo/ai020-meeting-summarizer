# Email Checklist

- Gebruik `scripts/build_emails.py` om CSS inline te zetten.
- Controleer de output in `emails/out/`.
- Gmail: controleer knoppen, `<details>` fallback en linkkleuren.
- Outlook: controleer tabelbreedte, padding en serif heading rendering.
- Zonder `MAILGUN_API_KEY` schrijft AI020 HTML-bestanden naar `emails/out/` voor lokale inspectie.
